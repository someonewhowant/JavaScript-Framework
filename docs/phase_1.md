# Jiggle.js — Документация Фазы 1 (Reactive Core)

В рамках первой фазы был полностью переписан механизм реактивности. Вместо устаревшего и ограниченного подхода на базе `Object.defineProperty` внедрён современный подход на базе `Proxy` (аналогично Vue 3).

## 1. Реактивность объектов (`reactive`)

Функция `reactive` теперь использует `Proxy`. Это решает сразу несколько ключевых проблем:
- **Вложенность:** Реактивность теперь работает рекурсивно. При обращении к вложенному объекту (например, `state.user.firstName`) возвращается реактивная обёртка этого вложенного объекта.
- **Новые свойства:** Динамическое добавление или удаление свойств из объекта теперь корректно перехватывается.
- **Массивы:** Поддержка операций над массивами (`push`, `splice` и т.д.).

```javascript
const state = reactive({
  user: {
    firstName: 'John',
    lastName: 'Doe'
  },
  age: 30
});
```

## 2. Примитивы (`ref`)

Для оборачивания примитивных значений (чисел, строк, булевых значений) введена функция `ref`. Она возвращает объект со свойством `value`.

```javascript
const count = ref(0);
console.log(count.value); // 0
count.value++;            // Триггерит эффекты
```

## 3. Вычисляемые свойства (`computed`)

Функция `computed` позволяет создавать свойства, значение которых зависит от других реактивных данных. 
Текущая реализация автоматически обновляет значение при изменении зависимостей функции-геттера.

```javascript
const fullName = computed(() => `${state.user.firstName} ${state.user.lastName}`);
```

## 4. Наблюдатели (`watch` и `watchEffect`)

Для выполнения побочных эффектов при изменении конкретных данных добавлены методы наблюдения:

### watchEffect
Работает так же, как и `createEffect` — немедленно выполняет переданную функцию, собирает её зависимости, и перезапускает функцию каждый раз, когда меняется хотя бы одна из этих зависимостей.

```javascript
watchEffect(() => {
  console.log('Текущий возраст:', state.age);
});
```

### watch
Принимает конкретный источник и функцию обратного вызова. Вызывается только при изменении источника.

```javascript
watch(
  () => state.age, 
  (newVal, oldVal) => {
    console.log(`Возраст изменился с ${oldVal} на ${newVal}`);
  }
);
```
