# Jiggle.js — Документация Фазы 5.3 (Plugin API)

Для того чтобы фреймворк был расширяемым, мы внедрили полноценный **Plugin API** и перешли на концепцию **Application Instance** (Экземпляр приложения), полностью повторяя архитектурный подход современного Vue 3.

## 1. Функция `createApp`

Вместо глобальной функции `render`, которая просто отрисовывает VNode в контейнер, теперь используется `createApp`. Это позволяет создавать независимые экземпляры приложения, у каждого из которых могут быть свои собственные плагины, миксины и внедрённые зависимости.

```javascript
import { createApp } from './jiggle.js';
import App from './App.js';

const app = createApp(App);
app.mount('#app');
```

## 2. Разработка Плагинов

Плагин — это объект с методом `install` или просто функция. При вызове `app.use(plugin, options)` метод `install` получает доступ к экземпляру приложения.

### Возможности плагинов:
1. **Глобальные миксины (`app.mixin`)**: Позволяют добавлять хуки жизненного цикла во **все** компоненты приложения.
2. **Внедрение зависимостей (`app.provide`)**: Позволяет предоставлять сервисы, доступные в любом компоненте дерева через функцию `inject()`.

```javascript
const MyPlugin = {
  install(app, options) {
    // 1. Добавляем логирование при монтировании каждого компонента
    app.mixin({
      onMounted() {
        console.log('Компонент был смонтирован!');
      }
    });

    // 2. Предоставляем глобальный сервис
    const apiService = new ApiService(options.baseUrl);
    app.provide('api', apiService);
  }
};

// Подключение:
app.use(MyPlugin, { baseUrl: 'https://api.example.com' });
```

## 3. Использование `inject`

Любой компонент в дереве может получить доступ к внедрённым зависимостям через `inject`.

```javascript
import { inject } from './jiggle.js';

function UserProfile() {
  // Получаем доступ к API сервису, внедрённому плагином
  const api = inject('api');

  return () => h('div', null, 'User Data Loader');
}
```

С введением этой фазы, экосистема Jiggle.js становится безгранично расширяемой: теперь вы можете легко писать плагины для валидации форм, интернационализации (i18n), кастомных логгеров и многого другого!
